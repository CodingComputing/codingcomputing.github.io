The `info` command allows you view a command's Info documentation.

The Info documentation is usually more detailed than the `man` documentation.

The Info reader is slightly more feature-rich than the `less` pager that man uses.


## Usage

```
info [cmd]
```
Displays the Info documentation of [cmd], in an Info viewer.

Press `H` to get more help about shortcuts in the Info viewer.

Note that some commands may have a man-page and not an info page, while for others it may be vice-versa.


## Experiment

Type `info info` to open the Info documentation of `info`.

Type `H` and view the shortcuts of the Info viewer.
Use them to navigate.

Explore the hyperlinks in the Info pages.

Take a look at the Info pages of other commands you know.
Compare with the corresponding man pages.


## Conclusion

This command allows you to view detailed Info documentation of commands.

It is useful to get in-depth information about a particular command.
